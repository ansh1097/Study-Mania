WHILE LOOP

In C++, while loop is used to iterate a part of the program several times. If the number of iteration is not fixed, it is recommended to use while loop than for loop. while loop can be address as an entry control loop. It is completed in 3 steps-

1. Variable initialization.( e.g int x=0; )

2. condition( e.g while( x<=10) )

3. Variable increment or decrement ( x++ or x-- or x=x+2 )

Basic Syntax is:

variable initialization ;

while (condition)

{

statements;

variable increment or decrement ;

}

C++ While Loop Example

Let's see a simple example of while loop to print table of 1.

#include

using namespace std;

int main () {

int i=1;

while(i<=10)

{

cout<<'i';

i++;

}

}

C++ Nested While Loop Example

In C++, we can use while loop inside another while loop, it is known as nested while loop. The nested while loop is executed fully when outer loop is executed once.

Let's see a simple example of nested while loop in C++ programming language.

#include

using namespace std;

int main () {

int i=1;

while(i<=3)

{

int j=1;

while(j<=3)

{

cout<<'i'<<''<<'j';

j++;

}

i++;

}

}

C++ Infinitive While Loop Example

We can also create infinite while loop by passing true as the test condition.

#include

using namespace std;

int main () {

while(true)

{

cout<<"Infinitive While Loop";

}

}
